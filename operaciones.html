Operaciones

Principales Operaciones con Cadenas

1. Concatenación

La concatenación es la operación que permite unir dos o más cadenas de caracteres en una sola. En muchos lenguajes de programación, se realiza utilizando el operador + o funciones específicas.

2. Substring (Subcadena)

La operación de substring permite extraer una porción específica de una cadena, definida por una posición inicial y una longitud determinada.

3. Búsqueda

La búsqueda consiste en localizar la posición de una subcadena dentro de otra cadena. Esta operación es fundamental para tareas como validaciones y análisis de texto.

4. Reemplazo

El reemplazo permite sustituir todas las apariciones de una subcadena por otra dentro de una cadena principal.

5. Conversión de Mayúsculas y Minúsculas

Esta operación permite transformar todos los caracteres de una cadena a mayúsculas o minúsculas, según sea necesario.

6. División (Split)

La operación de división o split consiste en separar una cadena en múltiples subcadenas basándose en un delimitador específico, como un espacio o una coma.

Ejemplo en JavaScript

            
        mensaje = "Hola"
        console.log(mensaje.split('')); // ["H", "o", "l", "a" ]
