Ejemplos

En este apartado, mostramos dos ejemplos de programas que implementan el manejo de cadenas (strings) en diferentes lenguajes.

Ejemplo 1: Inversión de una Cadena (JavaScript)

Este programa invierte una cadena ingresada por el usuario.


function invertirCadena(inputString) {
    // Convierte la cadena en un array, invierte el orden y vuelve a unir
    return inputString.split('').reverse().join('');
}

Ejemplo 2: Contar la Frecuencia de Palabras (JavaScript)

Este programa cuenta cuántas veces aparece cada palabra en una cadena.


function contarFrecuenciaPalabras(inputString) {
    // Convertir a minúsculas y eliminar signos de puntuación
    const texto = inputString.toLowerCase().replace(/[.,\/#!$%\^&\*;:{}=\-_`~()]/g, "");
    
    // Dividir en palabras
    const palabras = texto.split(/\s+/);
    
    // Contar frecuencia
    const frecuencia = {};
    for (let palabra of palabras) {
        if (palabra) { // Ignorar cadenas vacías
            frecuencia[palabra] = (frecuencia[palabra] || 0) + 1;
        }
    }
    
    // Convertir a formato de salida
    let resultado = "";
    for (let palabra in frecuencia) {
        resultado += `"${palabra}": ${frecuencia[palabra]} veces\n`;
    }
    
    return resultado;
}
